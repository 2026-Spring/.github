## 📚 학습 방식

- 학습 기록은 **TIL (Today I Learned)** 형식으로 진행합니다.
- 모든 학습 정리는 **Markdown (.md)** 파일로 작성합니다.
- 각자 이해한 내용을 **자신의 언어로 정리**하는 것을 원칙으로 합니다.
- 스터디 기간 동안 **GPT 사용을 지양**합니다.
- 스스로 고민하고 설명할 수 있는 수준의 이해를 목표로 합니다.

<br />

## 🔍 스터디 규칙

- 결석은 반드시 사전 공유합니다.
- 무단 결석 또는 반복적인 미참여 시 스터디 진행에 제한이 있을 수 있습니다.
- 질문은 언제든 환영하며, 질문 없는 회차를 목표로 하지 않습니다.
